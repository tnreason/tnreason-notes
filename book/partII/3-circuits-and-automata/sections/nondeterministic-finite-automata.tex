\section{Nondeterministic Finite Automata}

Instead of transition function we have in NFAs a transition relation.
Instead of directed boolean TT cores, we thus modify to generic boolean TT cores.

Usage:
\begin{itemize}
    \item Proof that any NFA is a DFA: By using basis encodings of the cores understood as linear maps.
\end{itemize}

\subsection{Stochastic Finite Automata}

Ideas:
\begin{itemize}
    \item The transition function is replaced by a cpd, which is encoded by a more general directed tensor and copied to all cores of a $\ttformat$.
    \item The result is a Markov Chain, i.e. a Bayesian Network on a chain graph.
    \item The contraction above then gives the probability of acceptance for a given input word.
    \item One can add a distribution over the input words, and contract to get a joint distribution of input words with an acceptance variable.
\end{itemize}
